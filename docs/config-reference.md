# Config Reference

The dependency-free v1 parser supports flat JSON key/value pairs. Required: `schema_version=qp.backtest.config.v1`, `data.path`. Common keys: `run.id`, `run.output_dir`, `portfolio.initial_cash`, `strategy.buy_threshold`, `strategy.sell_threshold`, `execution.fee_per_trade`, `execution.slippage_bps`.
